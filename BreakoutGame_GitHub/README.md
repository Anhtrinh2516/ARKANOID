# Breakout Game 
 
Enhanced Breakout game with multiple features. 
 
## How to run: 
1. Make sure Java is installed 
2. Double click `run_game.bat` 
3. Or run: `java -jar BreakoutGame.jar` 
 
## Features: 
- Classic Breakout gameplay 
- Skin shop system 
- Event system 
- Mini dice game
